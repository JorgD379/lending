YOLO для детектора СИЗ: кейс БГК

Февраль 2026 · В разработке

Задача: автоматизировать контроль соблюдения требований по средствам индивидуальной защиты (каска, спецодежда, обувь) на объектах ООО «БГК». Ручная проверка видео с нагрудных камер занимала часы и не давала структурированной отчётности.

Подход

Использовали YOLO для детекции объектов (каска, роба, штаны) и классификации нарушений. Обработка видео в реальном времени с формированием отчёта по временным меткам: когда и какой тип нарушения зафиксирован.

Результат

Автоматический отчёт по нарушениям СИЗ с привязкой к временным отрезкам. Сокращение времени на проверку в разы. Интеграция с существующими системами учёта.
